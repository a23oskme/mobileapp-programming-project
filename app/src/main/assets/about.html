Fågellistan

Målgruppen för 'Fågellistan' skulle kunna vara folk som gillar att fågelskåda som hobby eller ornitologer. Dessa personer skulle kunna lägga till egen data om fåglar dom sett med lite olika data, som plats eller storlek t.ex. För tillfället kan man dock inte lägga till data direkt i appen. Men om man skulle arbeta vidare med appen så skulle man kunna lägga till det och möjlighet att lägga till bilder.
